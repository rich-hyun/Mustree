MST 담보형 PoA

협력사가 늘수록
더 많은 MST가 잠기는 구조

무담보 발행은 없습니다. C=10 담보 규칙과 다운사이드 감액 금지로 네트워크 전체 총 스테이킹 목표 30,000,000 MST를 달성합니다.

전용망 만들기 정책 보기 브릿지 열기
총발행 MST 100,000,000 글로벌 담보비율 C=10 월 사용 상한 = 예치 MST × r / C GT 2,000개/체인 (1,000·1,000)

정책 핵심

1) 담보·발행 원칙

MST 담보로만 파트너 토큰(PT)을 발행합니다. 무담보 발행 없음.

PT = 예치 MST × r / C C(담보비율) = 10 r: A/B=100, C/D/E=500

2) 사용 상한 & 리필

월간 사용 상한은 발행식과 동일. 더 쓰려면 추가 예치가 필요합니다.

월 상한(PT) = 예치 MST × r / C 추가 예치 MST = 추가 PT × C / r

3) 락업·수수료

모든 수수료/한도/프리미엄은 MST로만 결제. 사용된 PT는 자동 소각.

다운사이드 감액 금지(Non‑Decreasing Collateral): 정책 변경으로 필요 담보가 낮아지더라도 기존 예치는 유지됩니다.

거버넌스(GT)는 체인당 2,000개(우리 1,000 · 협력사 1,000). 밸리데이터/업그레이드/비상정지 등 운영 권한에만 사용되며 가스/결제 기능은 없습니다.
