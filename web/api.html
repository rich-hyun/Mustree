머스트리 네트워크
API 제공 서비스

Mustree는 2025년부터 블록체인 기반 AI 융합 서비스를 위해
스마트 컨트랙트 기반 API를 제공하고 있습니다. 금융기관 및 엔터프라이즈 환경을 위한
안전한 온체인 API 게이트웨이를 제공합니다.

API 제공 이력

2025.09 — 블록체인 기반 AI 융합 응용서비스(API) 개발자금 지원사업

  • 주관: 한국인터넷진흥원(KISA)
  • 사용 기업: 국민은행
  • 제공 범위: 실물자산(RWA)·권리관리·담보 기반 API 제공
  • 2025 블록체인 기반 AI 융합 응용서비스(API) 개발자금 지원사업 참여
  • 블록체인 기반 금융시스템 개발 및 시범도입 지원

제공 가능한 API

Mustree API 게이트웨이는 NFT, SBT, RWA, 토큰 정책, 브릿지, 거버넌스, 노드 관리 등 엔터프라이즈 Web3 전반을 아우르는 API를 제공합니다.
