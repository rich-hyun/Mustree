API User Guide

스마트 컨트랙트 기반
API 사용 설명법

복잡한 스마트 컨트랙트 호출을 일반 HTTP API 방식으로 누구나 쉽게 사용할 수 있도록 구성된 가이드입니다.

OpenAPI 3.1 Auth: x-api-key Network: 사내 PoA/테스트넷

초보자를 위한 API 설명 요약

이 섹션은 MST-Contract API를 처음 사용하는 분들을 위해 스마트 컨트랙트를 어떻게 사용하면 좋은지를 자연스러운 문장으로 쉽게 정리한 안내입니다.

1. MST-Contract API란?

블록체인 상의 스마트 컨트랙트를 직접 다루지 않아도, 일반적인 웹 API 호출만으로 NFT 발행, SBT 발급, RWA 등록과 같은 기능을 사용할 수 있게 만든 서비스입니다. 지갑 연결, 가스 설정, 서명 처리 등 복잡한 과정은 모두 서버가 대신 처리합니다.

2. 어떻게 동작하나요?

여러분은 단순히 JSON 형태의 API 요청만 보냅니다. 그러면 서버가 해당 요청에 맞는 스마트 컨트랙트 함수를 자동으로 실행합니다. 트랜잭션이 블록체인에 기록되면 그 결과를 API 응답으로 전달합니다.

즉, “블록체인 사용 = 서버에 API 요청 → 서버가 대신 처리 → 블록체인 기록” 이 구조라고 이해하면 됩니다.

3. 사용 절차 (간단 요약)

  1. ① API 키를 발급받습니다. (x-api-key)
  2. ② Base URL을 확인합니다. 예: https://api.mst.network/v1
  3. ③ 필요한 기능(NFT/SBT/RWA 등)에 맞는 API를 호출합니다.
  4. ④ 응답으로 전달되는 transactionHash는 “블록체인 기록 증거”입니다.
  5. ⑤ 관리 대시보드에서 처리 상태와 로그를 확인합니다.

4. 예시 문장으로 이해하기

“NFT를 발행한다” → POST /v1/nfts/mint 를 호출하면 서버가 블록체인에 NFT를 생성하는 트랜잭션을 자동으로 기록합니다.

“실물자산을 블록체인에 등록한다” → /v1/rwa/mint API를 호출하면 해당 자산을 나타내는 RWA 토큰이 생성되고 기록됩니다.

“권리를 부여한다” → SBT API를 사용하면 사용자에게 양도 불가한 인증/권한 토큰을 발급할 수 있습니다.

즉, 스마트 컨트랙트를 직접 코드로 작성하거나 배포할 필요 없이 기존 REST API 쓰듯이 블록체인을 활용할 수 있습니다.
